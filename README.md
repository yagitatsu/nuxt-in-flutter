# nuxt-in-flutter

sample code of nuxt app in flutter web app

## Run

### 1. run nuxt app

```bash
$ cd nuxt
$ npm run dev
```

### 2. run flutter app

```bash
$ cd flutter
$ flutter run -d chrome
```

localhostのポートをいじる必要があるかも
